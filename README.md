# Creative CSS Card Effect

A modern multi-layer card design built using HTML5 and CSS3. This project demonstrates advanced CSS styling techniques, layered layouts, hover interactions, and responsive design principles.

##  Features

- Clean and modern card design
- Multi-layer visual effect
- Responsive layout
- CSS positioning techniques
- Hover animations and transitions
- Beginner-friendly code structure

##  Technologies Used

- HTML5
- CSS3

##  What I Learned

Through this project, I practiced:

- CSS Positioning
- Layer Management
- Flexbox Layout
- Responsive Design
- Hover Effects & Transitions
- UI Component Development

##  Purpose

This project was created to strengthen frontend development skills and explore creative UI design patterns using pure HTML and CSS.
